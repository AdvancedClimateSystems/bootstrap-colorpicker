    (static) enableHexAlpha :boolean

    This only applies when the color format is hexadecimal. If true, the alpha channel will be allowed for hexadecimal formatted colors, making them having 4 or 8 chars (RGBA or RRGGBBAA). This format is not yet supported in IE/Edge browsers, so it is disabled by default. If false, rgba will be used whenever there is an alpha change different than 1 and the color format is automatic.

    Default Value:
    • false

    (static) useAlpha :boolean

    If true, the alpha channel bar will be displayed no matter what.

    If false, it will be always hidden and alpha channel will be disabled also programmatically, meaning that the selected or typed color will be always opaque.

    If null, the alpha channel will be automatically disabled/enabled depending if the initial color format supports alpha or not.

    Default Value:
    • true

    (static) useHashPrefix :boolean
